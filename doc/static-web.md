# static-web

前端页面： 主要负责页面的展示， 逻辑交互

# 1. 登录页面

实现用户的登录功能

1.  获取用户输入的：  `用户名` ， `密码` 
2. 发送Ajax请求， 登录
3. 登录失败提示失败，成功则跳转后台首页

# 4. 操作项目环境页

显示 `变更名称` ，工程列表

## 4.1 工程列表

显示列表内容如下：

- 工程名： 工程名称

- 工程状态：-1:环境创建中， 0:待操作，1:打包中，2:部署中，3:运行中，4:打包失败，5:部署失败，6:测试通过，7:master代码有更新

- 操作按钮：

  master代码无需更新：

  1. 部署： 打包并且部署项目
  2. 测试通过： 测试通过，具有预发操作权限
  3. 回滚： 回滚项目至上一个版本
  4. 查看部署日志： 查看部署日志

  master代码需要更新：

  - 合并代码

# 5. 操作预发环境页

  - 无操作权限
